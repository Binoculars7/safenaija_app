## 🏆 Why This Project Matters (Hackathon Pitch)

🚑 In emergencies, **seconds matter**—but many Nigerians don’t know *who to call*.

SafeNaija eliminates confusion by using **AI to understand incidents and instantly route them to the right agency**.

**Impact:**
- ⚡ Faster emergency response
- 🛡️ Reduced loss of lives & property
- 🇳🇬 Unified reporting system across Nigeria

## 🚀 Key Innovation

### 🤖 AI-Powered Incident Understanding
- Interprets **messy, real-world Nigerian text** (including pidgin)
- Works without complex datasets using **LLM classification**

### 🔄 Smart Emergency Routing
- Automatically sends reports to:
  - 🚓 Police
  - 🚒 Fire Service
  - 🚑 FRSC / Emergency responders

### 📍 Location-Aware Decisions
- Uses **State + LGA** to route accurately

---

## ⚙️ How It Works (Simple Flow)

1. **Register (Demo NIN)**  
2. **Make Demo Payment**  
3. **Describe Incident**  
4. **AI Classifies It**  
5. **System Routes to Correct Agency**  

✅ No need to know who to call  
✅ No delays  
✅ Fully automated

## 🏗️ System Architecture

- **Frontend:** User reporting interface  
- **Backend:** Logic + routing engine  
- **AI Layer:** Incident classification  
- **Database:** Agency mapping (State + LGA + Type)  
- **Notification:** Email & SMS dispatch
